Step 1 of 5: Introduction

Welcome to the corrected breakdown of 2025 DSE Math Q18, following the official marking scheme.

"Let g(x) = 3x² - 6kx + 24x + 3k² - 24k + 55, where k > 0."

Key concepts we will use:

  • Vertex R: Found via completing the square.
  • Vertex S: Found via translation (Right 6, Down 10).
  • Point T: Origin O(0,0) is the orthocentre.
  • Final Goal: Find k using concyclic properties.

Part (a): Coordinates of R

Completing the square for g(x):

= 3(x² - (2k - 8)x) + 3k² - 24k + 55

= 3[x² - 2(k - 4)x + (k - 4)² - (k - 4)²] + ...

= 3(x - (k - 4))² - 3(k - 4)² + 3k² - 24k + 55

= 3(x - k + 4)² + 7

Coordinates of vertex R = (k - 4, 7)

Part (b)(i): Coordinates of T

1. S = (k + 2, -3) (R translated 6 right, 10 down).

2. OT ⊥ RS: Slope RS = -5/3. Thus, Slope OT = 3/5. Let T = (5t, 3t).

3. RT ⊥ OS: Slope OS = -3 / (k + 2).

// Orthocentre perpendicularity eq:

((3t - 7) / (5t - (k - 4))) * (-3 / (k + 2)) = -1

t = (k² - 2k - 29) / (5k + 1)

T = ( 5t, 3t )

Part (b)(ii): Geometric Insight

Given S, T, U, V are concyclic and TV ⊥ RS (∠SVT = 90°).

Theorem: Since ∠SVT = 90°, TS must be the diameter. Therefore, ∠SUT = 90° (angles in semi-circle).

Final Calculation

Slope SU = -8 / (k + 7)

Slope TU = (3k² - 31k - 92) / (5k² + 15k - 140)

Equation: (Slope SU) × (Slope TU) = -1

Expanding leads to:

5k³ + 26k² + 213k - 244 = 0

(k - 1)(5k² + 31k + 244) = 0

k = 1
